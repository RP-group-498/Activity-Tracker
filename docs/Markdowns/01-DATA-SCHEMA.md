# Data Schema Reference

## Purpose
Defines all data structures used by the browser extension. Use this file when working on data capture, storage, or transmission features.

---

## 1. Activity Event (Single Page Visit)

This is the core data unit captured for each page visit:

```typescript
interface ActivityEvent {
  // Identifiers
  eventId: string;          // UUID generated by extension
  sessionId: string | null; // From desktop app, null if disconnected
  
  // Source
  source: 'browser';        // Always 'browser' for extension data
  activityType: 'webpage';  // For browser, always 'webpage'
  
  // Timestamps
  timestamp: string;        // ISO 8601 format: "2025-01-15T10:30:00.000Z"
  startTime: string;        // When page became active
  endTime: string | null;   // When user left page (null if still active)
  
  // Page Data
  url: string;              // Full URL including path
  domain: string;           // Extracted domain (e.g., "youtube.com")
  path: string;             // URL path (e.g., "/watch")
  title: string;            // Page title from document.title
  
  // Metrics
  activeTime: number;       // Seconds user was active on page
  idleTime: number;         // Seconds page was open but user idle
  
  // Context
  tabId: number;            // Chrome tab ID
  windowId: number;         // Chrome window ID
  isIncognito: boolean;     // From incognito window (if permitted)
  
  // Sync Status
  synced: boolean;          // Has been sent to desktop app
  syncAttempts: number;     // Number of failed sync attempts
}
```

```json
}
```

## 2. Tab Switch Event

Captures when user switches between tabs:

```typescript
interface TabSwitchEvent {
  eventId: string;
  sessionId: string | null;
  timestamp: string;
  
  fromTabId: number;
  fromUrl: string;
  fromDomain: string;
  
  toTabId: number;
  toUrl: string;
  toDomain: string;
  
  synced: boolean;
}
```

## 3. Idle State Change

Tracks when user becomes idle or returns:

```typescript
interface IdleStateEvent {
  eventId: string;
  sessionId: string | null;
  timestamp: string;
  
  previousState: 'active' | 'idle' | 'locked';
  newState: 'active' | 'idle' | 'locked';
  
  // Context at time of state change
  activeTabId: number | null;
  activeUrl: string | null;
  
  synced: boolean;
}
```

## 4. Native Messaging Payloads

### Extension → Desktop App

```typescript
// Activity data batch
interface ActivityBatchMessage {
  type: 'activity_batch';
  events: ActivityEvent[];
  extensionVersion: string;
}

// Connection request
interface ConnectMessage {
  type: 'connect';
  extensionId: string;
  extensionVersion: string;
}

// Heartbeat
interface HeartbeatMessage {
  type: 'heartbeat';
  timestamp: string;
  pendingEvents: number;
}
```

### Desktop App → Extension

```typescript
// Session info
interface SessionMessage {
  type: 'session';
  sessionId: string;
  userId: string;
  status: 'active' | 'paused' | 'ended';
}

// Acknowledgment
interface AckMessage {
  type: 'ack';
  receivedEventIds: string[];
}

// Command
interface CommandMessage {
  type: 'command';
  command: 'pause' | 'resume' | 'clear_local';
}
```

## 5. Local Storage Schema

Data stored in `chrome.storage.local`:

```typescript
interface LocalStorageSchema {
  // Pending events not yet synced
  pendingEvents: ActivityEvent[];
  
  // Current session from desktop app
  currentSession: {
    sessionId: string;
    userId: string;
    startTime: string;
  } | null;
  
  // Extension state
  extensionState: {
    isConnected: boolean;
    isPaused: boolean;
    lastSyncTime: string | null;
  };
  
  // User preferences
  preferences: {
    trackIncognito: boolean;
    excludedDomains: string[];
    idleThresholdSeconds: number;
  };
  
  // Stats
  stats: {
    totalEventsCapture: number;
    totalEventsSynced: number;
    lastError: string | null;
  };
}
```

## Helper Functions Needed

```javascript
// Generate unique event ID
function generateEventId() {
  return 'evt_' + crypto.randomUUID().slice(0, 8);
}

// Extract domain from URL
function extractDomain(url) {
  try {
    const urlObj = new URL(url);
    return urlObj.hostname.replace('www.', '');
  } catch {
    return '';
  }
}

// Extract path from URL
function extractPath(url) {
  try {
    return new URL(url).pathname;
  } catch {
    return '';
  }
}

// Format timestamp
function getTimestamp() {
  return new Date().toISOString();
}
```

## Storage Limits
- `chrome.storage.local`: 10MB limit
- Buffer max 1000 events locally before forcing sync attempt
- Clear synced events immediately after desktop app ACK
